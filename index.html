Evolution of websites

  • All of the aproaches have their pros & cons
  • All are in use today
  • All may be appropriate depending on the use case

Static websites

  • Simply styled HTML and CSS
  • Backend simply serves content doesn't do any logic
  • Good for hosting documents online (CVs, Books, Articles)

Generated websites

  • HTML is generated by the backend from templates and partials that conain logic
  • Forms submitted to the backend and state influence how the site displays
  • Simple and solid but can your have to refresh the page to get latest data

Asyncronous websites

  • HTML is served from the backend and modified on the client
  • Data is served asyncronously from the backend and updated on the client using js
  • Quickly gets messy, logic gets replicated on the server and client

Reactive websites

  • HTML (View) is generated on the client
  • HTML is re-rendered every time the state changes
  • State gets updated asyncronously

What is React

  • React is a JavaScript library created by Facebook
  • React is a User Interface (UI) library
  • React is a tool for building UI components

At it's core it's a library that takes some state and generates some HTML. It's all about components.

What is it good for

  • Organising code in complex websites into managable chunks
  • Writting reusable components
  • Updating website as soon as new data arrives

import React from "react";

function Placeholder(props) {
    const width = props.width;
    const height = props.height;
    const url = 
        "http://placekitten.com/g/" + width + "/" + height;

    return (
    <img 
        src={url} 
        alt={"Kitten width: " + width + "x height: " + height} 
        title="Kittens are lovely"  
    />
    );
}

export default Placeholder;

Component State


import React, { useState } from "react";

function Placeholder(props) {
    const state = useState(true);
    const isCat = state[0];
    const setIsCat = state[1];
    let site = "";

    if (isCat) {
        site = "http://placekitten.com/g/";
    } else {
        site = "https://www.placecage.com/";
    }

    const toggleImage = function() {
        setIsCat(!isCat);
    };

    const url = site + props.width + "/" + props.height;

    return <img onClick={toggleImage} src={url} />;
}

export default Placeholder;

Moar components!


import React, { useState } from "react";
import Placeholder from "./Placeholder";

function Photoboard() {
    const state = useState(
        [{ width: 10, height: 10 }, { width: 10, height: 10 },  { width: 10, height: 10 }]
    );
    const photos = state[0];
    const changePhotos = state[1];

    function grow() {
        photos.push(photos[0]);
        photos.map(function(val, index) {
            photos[index] = { width: val.width * 2, height: val.height * 2 };
    });

        changePhotos(photos);
    }

    return (
        <div>
            {photos.map(function(value) {
            return <Placeholder width={value.width} height={value.height} />;
            })}
            <button onClick={grow}>Grow</button>
        </div>
    );
}

export default Photoboard;
